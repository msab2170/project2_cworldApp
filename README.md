# project2_cworldApp
project_cworld를 간단한 안드로이드 어플로 변형

로그인 부분 코틀린으로 구현,
메뉴 대분류 및 소분류 fragment등을 통해 코틀린으로 구현
상품화면, 결제화면 등은 webview를 통해 project_cworld의 뷰 부분을 일부 수정하여 적용

코틀린 제외한 개발환경은 project_cworld와 동일.
